# Evidence directory

RC1 intentionally ships no unreviewed benchmark or ASIC result files. This
placeholder documents the public evidence boundary so that a future fresh-clone
run can add profile-scoped summaries, manifests and checksums without changing
the source layout.

Only evidence with a readable raw log, exact source commit, profile/config
identity, input hash and reproduction command may be promoted to a public
verified claim. Private workbench paths, PDKs, libraries and generated tool
databases must never be copied here.
